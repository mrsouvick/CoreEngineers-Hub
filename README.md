# CoreEngineers Hub - EdTech Platform

A professional edtech platform built for MAKAUT engineering students, providing semester-wise learning resources across all core engineering branches.

## Features

- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Course Catalog**: Filter courses by branch and semester
- **Resource Library**: Download study materials, notes, and roadmaps
- **Professional UI**: Clean, modern interface with smooth animations
- **Team Showcase**: Dedicated section for team members

## Tech Stack

- React.js 18
- Tailwind CSS
- React Icons
- Vite (Build Tool)
